《崩捶篇》

筆者與同學們練習主要是三種拳法的應用，先後有：

  1. 插捶
  2. 拳擊[Boxing] - 現在講解的
  3. 崩捶[Back  Fist]

其實，四十多年前，約1978年間，筆者在尖沙咀青年會中心的健身房認識了一位來自美國的退伍軍人也正在練習，當時成哥，我與他閒談中得悉他練習過戰地武術，尤合軍練及東方武術，英文叫「Street Fighting」，當時中文翻譯為「搏殺術」，他們叫崩捶做「Defending Punch」，意思是既可防禦又可同時攻擊敵人，而崩捶與掛捶不同，前者以踭底力帶動腰馬，而後者膊，手帶動，而崩捶的快速攻擊的速度不下於Jab刺拳，甚至個別練習者其速度比 Jab 更快，不過在快之時能發揮攻擊敵人的勁度。
本篇崩捶的打法主要是参考北螳螂拳，略有参考我國的鷹爪拳，北勝蔡李佛，洪拳，詠春，及了解截拳道，也有潭[譚]腿的第 1、10、6、12 路類似打法及白鶴拳的「六力拳」的穿捶借鑑打法。

注要：初練者不宜太多練習，担心接駁肘關節附近的肌肉靭帶受傷，宜練習前做有关舒筋運動。

步法：在插捶篇已基本講述

<<打法>>
+ <A> 單對單
A1. 後護手，前[橫]崩
A2. 前抬高向下扲[伏]手，後[直]崩，後膝略屈曲
A3. 右搒手，左扲 / 擸手，崩捶
A4. 左伏 / 拍 / 外擋，略轉身右搒手，順勢右崩捶
A5. 凌空轉身鞭捶，但筆者在這個動作在年青時顯然印跟我練習武術的 「呂」 某，作大家參考。他的打法:
先移步與敵人的間距約3尺，而角度120至135度之間，在敵人進入陷阱的空間時，即凌空約270度轉身鞭捶攻其不備而遭擊倒 [在競技上用這些招式，未嘗不可]
+ <B> 超過一個攻擊敵人
B1. Jab，前橫崩
B2. Jab，向前逼步，扲[伏]手後直崩[略曲後腳]
B3. Jab[左]，左手收手在右邊頭部外[曲肘]，右後直拳，再左崩捶
B4. 大勾 / 細勾後轉崩
  1. 前手定步
  2. 前手移後腳內轉
  3. 前護手後橫勾再崩
B5. 側前上崩，側前下崩，後護手隨機應变
B6. 側前下崩，側前上崩，後護手隨機應变
B7. 前拍 / 扲 / 護被敵人來手攻擊，左手上崩捶，再右下崩插
B8. 前崩，摟手 / 拉手 蓄勁
  1. 後中正直捶
  2. 再連環直拳
B9. 前崩，摟手，後中正直崩
B10. 後扲 / 伏手阻敵攻擊，崩捶，左穿 / 擸手，右勾拳
B11. 後扲 / 伏手阻敵攻擊，崩捶，左穿 / 擸手，右勾手 [五指拼攏插眼 / 太陽穴]
B12. 前扲 / 伏手，護被敵攻擊，後直崩
  1. 轉移右步再右前崩
  2. 後崩後改為擸手將敵手下拉，再以前崩捶
B13. 参考太極肘底看捶，後搬手 / 拳，前崩，後窩肚捶
B14. 前橫崩
  1. 後撞拳，再右勾
  2. 後手按 / 疾 / 伏敵來拳，前手橫崩，繼左手日字衝捶[壓着敵人橋手的同時]攻擊敵人心房 / 肝，再右勾拳攻敵頭部
B15. 参考插捶篇 B6，先前锋手插捶，後手扲 / 按住敵人捉我的前手，再前崩捶
當敵人控制了我的插捶，我繼而崩捶，我則以另一手空脫敵人欲纒我使用的崩捶，我則穿手，繼而使用直肘攻擊他
B16. 先後撞拳，再 B15，但插捶是較低攻擊
[鞭捶打法]
B17. （鞭捶打法一）
左、右、左直拳，右後脚踭向後轉變成兩腳同一水平，施展右鞭捶180度，再左鞭捶180度 [移前半前步]
B18. （鞭捶打法二）
左、右直拳，左勾，右脚踭向後弧形180度，同時右鞭捶360度，[移前一步]
B19. （鞭捶打法三）
左企椿，左手拍，橫推，左腳急向左轉後，變成右手右脚以急轉鞭捶攻擊敵人[後退攻擊，出奇制勝，常有此法擊倒敵人]
+ <C> 崩捶加摔法
*因有崩捶關連的例如腿擊，鎖技都會有混合情況，因此，只會以優先编寫，而重覆在腿擊或破腿擊時，則只以編號列出，另也盡量用北螳螂拳的名詞[若我記得的話]，而有出於截拳道我又不知如何命名，只能描述打法而矣！
C1. 耕手，右前手搒貼敵人來腿攻擊，左後手下捲擸敵攻擊的脚，右崩捶，右扁踩，無論攻擊到敵人單脚站立與否。向右踏，身右 / 前壓至敵倒地…
C2. C1 攘面腿擊敵站立的脚脚脛 [上 / 下5寸]，右手搭敵頭 / 頸，拉直至敵倒地…
C3. 似 C1，但左手是轉下托住敵攻擊你的脚，右崩捶擊敵時，左手用力托高敵脚直至對方倒下…
C4. 右前崩，被敵右手擋隔，右手順勢微曲肘斜下拉，移步前腳跟貼敵脚跟同時以略轉腰左崩捶擊敵頭部，無論敵人是否擋隔你的左手與否，你順勢將右手向敵胸橫掃，加上你右腳移前控制敵人右脚，使他被逼倒下
C5. 我以右手索到敵人以右手攻擊我，將其手反至右肘向天，並以 [入環左叠肘] 壓力着敵人右肘時，我的右腳在敵人右腳前面，我順勢將我左腳剷後，使敵右腳扑下，而我左肘右手仍控制敵人的右手，對方必然甩肘關節…
C6. 與 C5 不同是我左腳在敵人右脚後，並以左膝控敵人右膝，對方右曲時我順勢左崩捶攻擊敵人頭部再順勢拉他右膊，加上左揪腿與否，敵必倒地…
C7. [撤步左/右躲剛] 敵以右直拳攻擊我中，上路，我則以移右脚至斜後方同時右橫崩擋開攻擊，並以左崩攻擊敵人，無論是穿過敵人橋手與否，我左腳以七星步勾腳移向左前方使敵人跌倒…
若敵人仍未跌倒，我則以左勾腳移略向近右腳，同時手前推倒敵人…
C8. 敵人左方向我，我以右崩捶攻敵人頭部，[但注意的是，敵人左手抬高至頸膊胸水平，那麼我左手以攤 / 扲 / 疾 / 伏先控對方的前鋒手]，無論對方是否能擋隔我的右手，我右拳順勢化為手拉敵胸 / 膊，而右脚以揪腿掃低對方…
C9. 右摏，右前崩捶攻敵頭部被擋隔，左手穿上高於我右手以控敵人截我右手，同時右手斜下以劈 / 割之勢打向敵人膝窩 / 小腿間，同時下蹲身體似以入環步配合，而我步已移貼或略過敵人前脚後面，再將斜上身，橫手向敵人身體，一蹬致令敵人被摔倒。

D8. 與D7前段相同，但右手可控制到敵來腿時 [可化刁手]，並以垂手崩捶 / 手攻擊敵人下陰

D10. 先以前脚扁踩攻擊敵人疑起脚的前鋒脚，再以前手橫崩捶打擊敵人頭部
+ <E> 加鎖技法
本篇不再提用指 / 腕 / 脚等鎖技
E1. C5 [入環左叠肘] 由上壓下敵人肘，而前脚在前
E2. C6 [入環左叠肘] 但前脚在後，也是下壓敵肘
E3. 右前崩捶，轉刁手擸敵人右手近手腕，拉正其肘窩向天，以拗步左抽踭，做成上托壓敵肘
E4. E3 再順勢左手按貼敵人右手肘附近防被其出踭，我右手弧形向敵人的頭部扣頸的同時左步橫行做成我左腳在敵左腳後面，而右手踏着我提起的左手橋手間，而我左手務求按着敵人頭後面，兩者我右手向我身拉近，而我在手則向前推，直至断頸
若 E3，而敵人右手下曲肘抵消我的肘托，我左手在上托的同時順勢跟移步近貼敵身衝踭下敵人腹側，若對方的肘弯下太低時，我則拍肘攻擊敵人腹的後面
E5. 敵与我對立，以右前步右前鋒手直拳攻我，我以左護手撞隔右崩捶攻敵右邊頭部，再以擸頸向下拉而右橫膝打敵人腹部，使其彎身，敵手兼顧我膝，我將右手扲低敵人頸，再扣以右橋手扣其頸，而右手搭着左手橋手，向我身拉近，而左手向前推，這招也有叫「斷頭台式前頸鎖」 若對方用力糾纏，我可將其中一隻脚向後伸踏，做成拖低敵人，容易做頸鎖
E6. [膊頭鎖] 當我崩捶攻擊或defence防禦，阻擋來拳時，化為刁手擸緊敵人橋手，有定位，順勢斜上右脚，敵的右手膊頭被鎖扭近90度以上，我再左斜步部後移，敵人膊頭受扭曲至270度以上，當場膊頭受傷

F15. 我站左椿，敵人向我胸或以上攻擊，我助低馬，像 [八段錦第五式搖頭擺尾去心火] 將身移閃退近右腳，再從右邊斜上身向下，差不多上身與腳盡量成90度，左手護頭頸直至搖至左邊擋隔，右手則以擋左
  1. 先吸捶再擂手崩捶攻擊敵人頭部，再右釘腿，或再加上彈腿，然後手腳繼續衝前
  2. 以批踭攻敵人下腭 / 耳 / 太陽穴再崩捶攻擊頭部，因與敵人距離比 1 較近，以釘腿踢敵人前腳，再以扁踩踢敵人後腳，[疑似12路潭腿的第6路]
F16. 仿效潭腿第12路，先以左假手高揚向敵人，右寸腿，再坐馬上前右橫中勾捶攻擊敵人下腹，步略移前的同時以崩或掛捶 [視乎情況] 打敵人頭部
F17. 前崩捶 [或左右虛式] 加前當門腿傷敵膝蓋
